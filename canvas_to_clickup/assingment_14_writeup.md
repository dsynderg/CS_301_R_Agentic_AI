# What I learned

- More often than not, AI agents could be replaced with actual code that would make the process faster and more efficent, as well as more cost efective.
- This is another time where AI is a hammer, and every problem isn't a nail. It would almost always be better to not have AI solve your problem code is always faster and cheaper.
